# Гармония - исходники сайта

Это статическая версия сайта проекта "Гармония" для передачи разработчикам.

## Состав

- `index.html` - структура страниц и контент.
- `styles.css` - все стили, адаптив, анимации и визуальные состояния.
- `script.js` - меню, модальные окна, слайдеры, переключение инфраструктуры.
- `assets/` - локальные изображения, логотипы и графика сайта.

## Как открыть

Можно открыть `index.html` напрямую в браузере.

Для корректной проверки как на сервере:

```bash
python -m http.server 8088
```

После запуска открыть:

```text
http://127.0.0.1:8088/
```

## Правки этой итерации

- Добавлены favicon (`assets/brand/favicon*.png`, `favicon.ico`) на основе фирменного знака.
- Добавлены Open Graph / Twitter meta-теги и `theme-color` для корректных превью при шаринге ссылки.
- Добавлен скрытый `<h1>` на главном экране (доступность и SEO без визуальных изменений).
- Все изображения ниже первого экрана получили `loading="lazy" decoding="async"`; hero-изображение получило `<link rel="preload">` для более быстрой отрисовки.
- `script.js` подключен через `defer`.
