# Гармония - исходники сайта

Это статическая версия сайта проекта "Гармония" для передачи разработчикам.

## Состав

- `index.html` - структура страниц и контент.
- `styles.css` - все стили, адаптив, анимации и визуальные состояния.
- `script.js` - меню, модальные окна, слайдеры, переключение инфраструктуры, курсор, скроллспай, лайтбокс.
- `assets/` - локальные изображения, логотипы, шрифты и графика сайта.

## Как открыть

Можно открыть `index.html` напрямую в браузере.

Для корректной проверки как на сервере:

```bash
python -m http.server 8088
```

После запуска открыть:

```text
http://127.0.0.1:8088/
```

## Правки этой итерации (редизайн "покруче")

- Самостоятельно хостятся шрифты `Cormorant Garamond` (заголовки, крупные цифры) и `Manrope` (текст) - `assets/fonts/*.woff2`, вариативные, с кириллицей.
- Кинематографичная зернистая текстура (`.grain`) поверх всего сайта.
- Кастомный курсор (точка + кольцо) с эффектом увеличения на интерактивных элементах и "магнитными" кнопками - только на устройствах с мышью и без `prefers-reduced-motion`.
- Полоса прогресса скролла в хедере и активная подсветка пункта меню (scrollspy) для текущего раздела.
- Вертикальная точечная навигация справа (десктоп) с подписями разделов при наведении.
- Заголовки разделов появляются по словам (stagger) при скролле.
- Цифры в блоках "О проекте" и "Локация" анимированно считаются при появлении в viewport.
- Бегущая строка (marquee) с названием проекта между первым экраном и блоком "О проекте".
- Полноэкранный лайтбокс для фотогалереи (клик/Enter, стрелки, Esc).
- Тёплое свечение (spotlight), следующее за курсором, на карточках фактов и на плитках инфраструктуры.
- Лёгкий параллакс логотипа в hero, анимированный индикатор скролла, эффект "блика" при наведении на кнопки.
- Исправлен баг в исходном макете: `backdrop-filter` на `.site-header` создавал containing block для `position: fixed` мобильного меню, из-за чего на телефонах меню открывалось не на весь экран, а сжатым в полосу хедера. Вынесли блюр в `::before`, меню теперь разворачивается на весь экран.
